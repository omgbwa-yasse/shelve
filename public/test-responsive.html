Design Responsive Appliqué

Le fichier resources/views/layouts/app.blade.php a été mis à jour avec un design responsive complet.

Breakpoints Implémentés

  • Desktop - >991px (Bleu)
  • Tablette - ≤991px (Jaune)
  • Mobile - ≤767px (Orange)
  • Petit Mobile - ≤480px (Rouge)
  • Landscape - hauteur ≤500px

Fonctionnalités Ajoutées

  • Sidebar mobile coulissant
  • Bouton toggle flottant
  • Navigation horizontale scrollable
  • Header responsive adaptatif
  • Touch improvements (44px min)
  • Auto-fermeture sur resize

Éléments CSS Responsive

Tablette (≤991px)
  • Sidebar position fixe
  • Navigation scrollable
  • Header compact
Mobile (≤767px)
  • Header ultra compact
  • Logo 30px
  • SAI masqué
Petit Mobile (≤480px)
  • Libellés masqués
  • Icônes uniquement
  • Sidebar 85% largeur

Fonctions JavaScript

toggleMobileSidebar()

Toggle le menu latéral avec animation, changement d'icône et lock du body scroll.

window.resize listener

Ferme automatiquement le sidebar quand la fenêtre dépasse 991px.

Comment tester ?
  1. Redimensionnez votre navigateur pour voir les changements
  2. Utilisez Chrome DevTools > Device Toolbar (F12 puis Ctrl+Shift+M)
  3. Testez sur un vrai appareil mobile/tablette
  4. Observez l'indicateur de breakpoint en haut à droite
