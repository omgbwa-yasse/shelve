ÉTAPE 2 - STRUCTURE DU TITRE
Appliquez la structure appropriée :

→ CAS SIMPLE (1 objet) :
   Objet. — Action : typologie documentaire. Dates

→ CAS COMPLEXE (2-3 objets) :
   Objet principal. — Premier élément : typologie (dates). Second élément : typologie (dates). Dates extrêmes

→ CAS TRÈS COMPLEXE (3+ objets) :
   Objet principal. — Sous-objet 1 : typologie (dates) ; sous-objet 2 : typologie (dates), autre typologie (dates). Dates extrêmes

ÉTAPE 3 - PONCTUATION NORMALISÉE
• Point-tiret (. —) après l'objet principal
• Virgule (,) entre éléments équivalents
• Point-virgule (;) entre actions différentes
• Deux points (:) avant la typologie
• Parenthèses () pour les dates intermédiaires

ÉTAPE 4 - VOCABULAIRE CONTRÔLÉ
Utilisez les termes précis :
• Typologies : registres, correspondance, procès-verbaux, plans, états, listes, rapports, dossiers, etc.
• Actions : construction, aménagement, création, attribution, surveillance, gestion, etc.
• Liaisons : "avec", "dont", "contient", "concerne", "en particulier", "notamment"
